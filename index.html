Parity Changes (Must Have)

  • 4. Free cancellation - Pull from partner data, prominent display
  • 5. Remove sq ft - Simplification, no effort needed

Exceed Changes (Differentiation)

  • 4. Expanded amenities - More amenity icons displayed
  • 5. Hover states - CSS only, easy implementation
